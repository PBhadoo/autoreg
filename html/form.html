Type of resource record:

See the explanation to choose the type, then fill the following list.

If you are requesting a NS record
Fill Name1...NameX and IP1...IPX respectively with the fully qualified domain names and (only if necessary) the IP addresses for your domain servers; put the primary at the top of the list. You don't need to fill all lines. IP addresses are only needed when the domain names for the servers are within the requested domain.
If you are requesting a MX (or wildcard-MX) record
Fill Name1..NameX. The names you supply must point to an address (A type) record. The lowest precedence MX should be listed first.
If you are requesting a CNAME record
Only fill Name1. It must exist and must not point to a CNAME record.
